## Overview

A single-page interactive website featuring a 6-round binary quiz that compresses your choices into one defining word, followed by an explorable word cloud and full company website.

## Features

- **Typing animation** intro with type-delete-retype effect
- **6-round quiz** with Values, Strengths, and Skills dimensions
- **64 unique result words** mapped to choice combinations
- **Narrative compression** animation with dimension breakdown
- **Interactive word cloud** with depth blur layers, magnetic cursor, click ripple, and hover tooltips
- **Full scrollable website** with About, Story, Process, Services, and Team sections
- **Scroll progress indicator** with section-based color changes
- **Share your word** clipboard integration
- **Mobile-responsive** with touch support for word cloud

## Setup

Open `index.html` in a browser. No build step required.

Fonts (CircularStd) must be in the same directory as `index.html`.
